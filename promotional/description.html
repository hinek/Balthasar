Help Balthasar to find its way back home. To do that, enter the gate at the end of each level. If something is in its way, try to change its color. Puzzle your way through this platformer.

This is the first game I publish, thank you for trying it. Please leave some feedback, so I can grow, thank you.

Controls

WASD or arrow keys to move and jump
Hold Ctrl to run
Space to use levers
